# 🚀 Netlify Drop 部署步驟

## 你現在要做的事

### 1. 打開 Netlify Drop 網站
在瀏覽器中訪問：
```
https://app.netlify.com/drop
```

### 2. 拖拽 www 資料夾
- 我已經幫你開啟了 `www` 資料夾
- 將整個資料夾拖拽到 Netlify 網頁上
- 或者選擇資料夾內的所有檔案一起拖拽

### 3. 等待上傳（約 30 秒）
Netlify 會自動處理並給你一個網址

### 4. 複製網址
網址格式會像這樣：`https://random-name.netlify.app`

### 5. 在手機上打開這個網址
然後按照指示安裝到主畫面

---

## 📱 手機安裝步驟

### Android（Chrome）
1. 打開網址
2. 點選單（⋮）→「新增至主畫面」
3. 完成！

### iPhone（Safari）
1. 打開網址
2. 點「分享」→「加入主畫面」
3. 完成！
