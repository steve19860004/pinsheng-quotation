# 品盛冷氣行報價單系統 - 使用說明

## 📱 應用程式簡介

品盛冷氣行專業空調報價單管理系統，提供完整的客戶資料管理、報價項目編輯、自動計算、PDF 匯出及歷史報價查詢功能。

**公司資訊：**
- 公司名稱：品盛冷氣行
- 統一編號：00550134

---

## ✨ 主要功能

### 1. 報價單管理
- ✅ 自動生成報價單編號（格式：Q20260202-1234）
- ✅ 設定報價日期
- ✅ 客戶資訊管理（名稱、電話、地址）
- ✅ 動態新增/刪除報價項目
- ✅ 自動計算金額（小計、營業稅 5%、總計）
- ✅ 備註欄位

### 2. 報價項目
每個項目包含：
- 序號（自動編號）
- 品名（空調設備名稱）
- 規格說明（型號、規格）
- 數量
- 單價
- 金額（自動計算）

### 3. 資料管理
- 💾 **儲存報價單** - 將報價單儲存到本地
- 📋 **歷史報價單** - 查看所有已儲存的報價單
- 🔍 **搜尋功能** - 依客戶名稱或報價單編號搜尋
- 🗑️ **刪除功能** - 刪除不需要的報價單

### 4. 匯出功能（新增）
- 🖨️ **列印** - 列印報價單
- 📄 **匯出 PDF** - 將報價單匯出為 PDF 檔案
- 💾 **匯出資料** - 備份所有報價資料為 JSON 檔案
- 📁 **匯入資料** - 從 JSON 檔案還原報價資料

## 📖 使用教學

### 新增報價單

1. 點擊右上角「➕ 新增報價單」按鈕
2. 系統會自動生成報價單編號和日期
3. 填寫客戶資訊：
   - **客戶名稱**（必填）
   - 聯絡電話
   - 安裝地址

### 新增報價項目

1. 點擊「➕ 新增項目」按鈕
2. 填寫項目資訊：
   - 品名：例如「冷暖氣機」
   - 規格說明：例如「Panasonic CS-K36BA2 5坪變頻」
   - 數量：例如「1」
   - 單價：例如「25000」
3. 金額會自動計算
4. 小計、營業稅、總計會自動更新

### 儲存報價單

1. 確認所有資料正確
2. 點擊「💾 儲存報價單」按鈕
3. 系統會顯示「報價單已儲存！」訊息

### 查看歷史報價單

1. 點擊「📋 歷史報價單」按鈕
2. 側邊欄會顯示所有已儲存的報價單
3. 使用搜尋框可以快速找到特定報價單
4. 點擊任一報價單即可載入編輯

### 匯出 PDF

1. 確保已填寫客戶名稱
2. 點擊「📄 匯出 PDF」按鈕
3. PDF 檔案會自動下載
4. 檔名格式：`品盛報價單_[客戶名稱]_[報價單編號].pdf`

### 資料備份

#### 匯出資料
1. 點擊右上角「💾 匯出資料」按鈕
2. JSON 檔案會自動下載
3. 檔名格式：`品盛報價資料_[日期].json`
4. 建議定期備份，防止資料遺失

#### 匯入資料
1. 點擊右上角「📁 匯入資料」按鈕
2. 選擇之前匯出的 JSON 檔案
3. 系統會提示：「匯入資料將會覆蓋現有的所有報價資料，確定要繼續嗎？」
4. 確認後，資料會被還原

### 列印報價單

1. 點擊「🖨️ 列印」按鈕
2. 瀏覽器會開啟列印預覽
3. 選擇印表機或儲存為 PDF
4. 列印時會自動隱藏操作按鈕，版面更整潔

## 💡 使用技巧

### 快速複製報價單
1. 開啟想要複製的歷史報價單
2. 修改客戶名稱和其他需要變更的資訊
3. 點擊「儲存」，系統會建立新的報價單

### 批次修改價格
1. 開啟報價單
2. 直接在表格中修改單價
3. 金額會即時更新
4. 記得點擊「儲存」保存變更

### 整理歷史報價單
1. 定期檢查歷史報價單
2. 刪除不需要的舊報價單
3. 使用「匯出資料」功能定期備份重要資料

---

## ⚠️ 注意事項

1. **資料儲存**
   - 所有資料儲存在瀏覽器的 LocalStorage 中
   - 清除瀏覽器資料會導致報價單遺失
   - **建議定期使用「匯出資料」功能備份**

2. **客戶名稱必填**
   - 儲存報價單和匯出 PDF 前必須填寫客戶名稱
   - 系統會提示「請輸入客戶名稱！」

3. **報價項目**
   - 至少需要一個報價項目才能儲存
   - 刪除項目後序號會自動重新編號

4. **匯入資料**
   - 匯入資料會**完全覆蓋**現有資料
   - 匯入前建議先匯出目前資料作為備份
